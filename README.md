Personal tool for cleaning up and making stats for my Spotify library. \
Project uses Spotify API for most operations and Genius API for downloading song lyrics.

Features:
* Searching for single / EP releases,
* Searching for duplicated songs and artists,
* Searching for cross-playlist duplicates,
* Merging playlists,
* Synchronizing playlists
* Batch downloading song lyrics,
* Making various useless stats for a playlist:\
	Number of songs / first letter of title\
	Number of songs released in a specific year\
	Song duration histogram\
	Most popular words in song titles\
	Most popular words in song lyrics
